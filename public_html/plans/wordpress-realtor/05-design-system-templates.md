# Этап 5. Дизайн-система и шаблоны

## Зачем
Зафиксировать визуальный язык сайта до массового наполнения.

## Что сделать
- UI-kit
- сетка
- типографика
- цвета
- кнопки, формы, карточки
- шаблоны страниц
- mobile-first верстка

## Owner

## Особый фокус

- карточки новостроек и вторички должны быть сравнительно удобны
- модуль подбора должен иметь место под баллы и комментарии

## Тесты этапа
- [ ] no horizontal scroll at 390px, 768px, 1440px widths
- [ ] key blocks render in editor without errors
- [ ] header, hero, CTA, and footer match design system
- [ ] templates look consistent across main pages
- [ ] hero/CTA visible and clear on first screen
