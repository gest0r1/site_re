# Assets

Папка для проектных ассетов дизайна сайта `дом-эксперт.рф`.

## Содержимое

- SVG-файлы из flat asset pack загружены как отдельные файлы и могут использоваться opencode напрямую.
- PNG-файлы и полный ZIP-пакет при необходимости должны быть добавлены через git/opencode как binary-файлы либо восстановлены из исходного asset pack.

## Как использовать через opencode

Если ассет отсутствует в WordPress Media Library, opencode может загрузить конкретный SVG:

```bash
wp media import документация/assets/01-brand-logo-horizontal-light.svg --post_title="01-brand-logo-horizontal-light"
```

После импорта нужно обновить:

```text
/документация/05-prod-media-map.md
```

## Важно

- Не подставлять случайные изображения вместо этих ассетов.
- Для SVG на сайте должен быть активен Safe SVG.
- Risk-иконки использовать только вместе с текстовым пояснением.
