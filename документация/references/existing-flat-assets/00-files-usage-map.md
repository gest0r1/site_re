# Куда использовать файлы

| Настройка сайта → Свойства сайта → Логотип | `01-brand-logo-horizontal-dark.svg` | Основной логотип на светлой шапке или в настройках сайта. |

| Mobile header / компактные места / fallback | `01-brand-logo-mark.svg` | Знак без длинного текста. |
| Шапка mobile | `02-navigation-menu.svg` | Иконка открытия меню, если требуется кастомная. |
| Поиск | `02-navigation-search.svg` | Поиск по каталогу, FAQ, глоссарию. |
| Главная → карточка «Продаю квартиру» | `03-path-seller-home.svg` | Иконка seller-сценария. |
| Главная → карточка «Покупаю квартиру» | `03-path-buyer-key.svg` | Иконка buyer-сценария. |
| Блок бесплатных инструментов → Оценка | `03-path-estimate.svg` | Оценка квартиры / расчет стоимости. |
| Блок бесплатных инструментов → Риск-чек | `03-path-risk-check.svg` | Проверка квартиры / юридические и объектные риски. |
| Сравнение объектов | `03-path-compare.svg` | Сравнение квартир, районов, параметров. |
| Карточка объекта → площадь | `04-property-area.svg` | Площадь квартиры. |
| Карточка объекта → комнаты | `04-property-rooms.svg` | Количество комнат. |
| Карточка объекта → этаж | `04-property-floor.svg` | Этаж / этажность. |
| Карточка объекта → адрес | `04-property-location.svg` | Адрес, район, метро. |
| Карточка объекта → избранное | `04-property-favorite.svg` | Добавить объект в избранное. |
| Статус объекта | `05-status-available.svg` | Доступен / проверен / активен. |
| Статус объекта | `05-status-sold.svg` | Продан. |
| Блок доверия / CTA эксперта | `06-trust-expert.svg` | Персональная экспертность. |
| Формы / CTA | `07-form-phone.svg` | Телефон, контакт, обратный звонок. |
| Risk summary | `08-risk-high.svg` | Высокий риск. Всегда сопровождать текстом. |
| Risk summary | `08-risk-medium.svg` | Средний риск. Всегда сопровождать текстом. |
| Risk summary | `08-risk-low.svg` | Низкий риск. Всегда сопровождать текстом. |
| Risk summary | `08-risk-verified.svg` | Проверено экспертом. |
| Файл favicon SVG | `10-favicon-favicon.svg` | Можно хранить как исходник favicon. |
| Site icon SVG | `10-favicon-site-icon-512.svg` | Исходник иконки 512. |
| Favicon 16 | `10-favicon-16.png` | Технический favicon 16×16. |
| Favicon 32 | `10-favicon-32.png` | Технический favicon 32×32. |
| Apple touch icon | `10-favicon-apple-touch-icon.png` | Иконка для iOS/закладок. |
| WordPress → Свойства сайта → Иконка сайта | `10-favicon-site-icon-512.png` | Основной файл для загрузки как иконка сайта. |
