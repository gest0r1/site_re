# Дом-Эксперт — flat asset pack для WordPress Media Library

Эта версия сделана без папок. Все файлы лежат в корне архива.

## Логика именования

Формат:

```text
NN-category-name.ext
```

Где `NN` — порядок группировки при сортировке по имени в медиатеке WordPress.

## Группы

```text
00 — документация
01 — brand / логотипы
02 — navigation / навигация
03 — path / сценарии и инструменты
04 — property / параметры объекта
05 — status / статусы объектов
06 — trust / доверие и экспертность
07 — form / формы и контакты
08 — risk / риски и проверки
09 — poster / крупные SVG-постеры
10 — favicon / иконки сайта
```

## Рекомендация по загрузке

Загружать можно все файлы сразу через:

```text
Медиафайлы → Добавить новый
```

После загрузки отсортируйте медиатеку по имени. Файлы будут сгруппированы по префиксам.

## Важно

- Для SVG должен быть активен Safe SVG.
- `10-favicon-site-icon-512.png` использовать как основную иконку сайта WordPress.
- SVG-постеры использовать как изображения/фоновые медиа в Kadence Row Layout или Gutenberg Cover.
- Risk-файлы не использовать без текстового описания уровня риска.
